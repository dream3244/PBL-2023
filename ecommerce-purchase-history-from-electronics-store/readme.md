## datasource： 
https://www.kaggle.com/datasets/mkechinov/ecommerce-purchase-history-from-electronics-store
### about the dataset:
* event_time -购买时间  
* event_type -行为类别  
* product_id -产品编号  
* category_id -产品的类别ID  
* category_code -产品的类别分类法（代码名称）  
* brand -品牌名称  
* price -产品价格  
* user_id -用户ID
